运行参数

SMTP 接收策略

这些参数会写入 system_settings，并在当前运行进程中立即生效。

最大邮件大小

单位为字节。超过此值的邮件会被拒收。

单封邮件最大收件人数

超出后该封邮件会被拒收。

SMTP 空闲超时（秒）

连接空闲超过此秒数会被服务器关闭。

SMTP 并发连接上限

设为 0 表示不限制并发连接数。

每 IP 短窗口连接上限

超过窗口阈值的 IP 会被暂时拒绝建立连接。

SMTP 连接限流窗口（秒）

配合上方"每 IP 上限"计算时间窗口。

磁盘告警阈值（%）

磁盘占用超过此百分比时会触发告警。
